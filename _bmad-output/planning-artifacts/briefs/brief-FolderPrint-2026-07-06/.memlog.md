- (decision) FolderPrint is a .NET 8 C# console application for registering trusted folder baselines and verifying file integrity over time.
- (decision) V1 stores SHA-256 file fingerprints in a local JSON catalog and compares current folder state against the trusted baseline.
- (decision) V1 command scope includes register, verify, list, unregister, duplicates, and refresh.
- (decision) V1 avoids GUI, SQLite, cloud integration, realtime monitoring, encryption, network shares, very large-scale optimization, and complex ignore rules.
