- (version) .NET 8 verified against Microsoft support policy on 2026-07-07: LTS maintenance support ends 2026-11-10; acceptable because user explicitly requested .NET 8.
- (decision) Use layered CLI/Core architecture: FolderPrint.Cli depends on FolderPrint.Core; Core never depends on CLI.
- (decision) Core owns scanning, hashing, comparison, duplicate detection, catalog persistence, and report data structures.
- (decision) CLI owns command parsing, argument validation, console output, and exit code mapping.
- (decision) Use human-inspectable JSON catalog at %AppData%\FolderPrint\catalog.json for V1; no SQLite or database.
- (decision) Use manual CLI parser for V1; System.CommandLine remains deferred because V1 command surface is small and minimal dependencies are preferred.
- (decision) Tests target Core business behavior primarily; CLI tests cover parser/exit-code mapping without coupling heavily to console text.
- (event) architecture spine finalized and copied project knowledge version to docs/architecture.md.
